The `date.year()` function returns the year of a specified time.

_**Function type:** Transformation_  

```js
import "date"

date.year(t: 2019-07-17T12:05:21.012Z)

// Returns 2019
```

## Parameters

### t
The time to operate on.

_**Data type:** Time_
